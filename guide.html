Comment payer

Via Wave CI
  1. Ouvre l'app Wave
  2. Envoie le montant exact au numéro marchand
  3. Note l'ID de transaction (ex: WA-AB12345)
  4. Saisis l'ID sur la page paiement
🟠 Via Orange Money
  1. Compose #150*1*1*NUMERO*MONTANT#
  2. Confirme avec ton PIN
  3. Note l'ID dans le SMS reçu
  4. Saisis l'ID sur la page paiement
